# 속초 여행 이벤트 페이지
<hr>
<h4>설명</h4>
<b>Postman</b>을 사용하여 <b>API</b>를 생성하였다. <br>
<b>API</b>에 저장된 <b>JSON</b>형태의 데이터를 <b>AXIOS</b>를 통해 받아 리액트에 적용히켰다.<br>
<h4>현재 진행상황</h4>
<b>Route</b>를 통해 페이지를 설정하는 과정이며 <b>AXIOS</b>를 통해 받아온 데이터를 원활하게 뿌려주는 데 약간의 차질이 생겼다.
